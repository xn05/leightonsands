# Leighton Sands

Minimal MonoGame framework scaffold with a title screen, sprite font, and content pipeline entries.

## Quick run

Build/run from your IDE, or with the .NET CLI if your MonoGame template supports it.

```
dotnet build
```

## Content notes

- Title texture: `Content/Textures/UI/titletext.png`
- Font: `Content/Font/main.ttf` via `Content/Font/Main.spritefont`
- Content pipeline config: `Content/Content.mgcb`
